# Edever

Edever是electron、antd、antv集成开发项目的命令行工具，用于项目模板的快速获取，将持续关注针对框架新特性的开发

## 如何使用

* 请在release中下载最新对应的可执行文件，只提供64位可执行文件

| 文件名 | 平台 |
| :---- | ---- |
| edever-win | Windows |
| edever-mac | MacOS |
| edever-linux | Linux |

* 对于Windows平台

> 局部命令, 在工程文件夹下

```shell
./edever -h
```

> 全局命令, 要求把工具所在目录添加到`Path`环境变量

```shell
edever -h
```
